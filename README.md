# 🧪 Selenium Automation Framework

## 🔹 Project Overview
This project is a Selenium-based automation framework designed to test an e-commerce website. It automates key user workflows such as registration, login, product selection, and checkout.

---

## 🔹 Key Features
- Cross-browser testing (Chrome, Firefox, Edge)
- Data-driven testing using CSV (OpenCSV)
- TestNG framework integration
- Extent Reports for execution analysis
- Screenshot capture on failure
- End-to-End (E2E) test flow
- Selenium 4 relative locators
- Explicit, Implicit & Fluent Waits

---

## 🔹 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- OpenCSV
- Extent Reports

---

## 🔹 Test Scenarios
- User Registration
- User Login
- Add to Cart
- Checkout
- E2E Flow (Login → Cart → Checkout)
